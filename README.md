# p5-boilerplate

Boilerplate for writing p5 sketches in es6

## Clone / Install

replace `project-name` with desired folder name

```bash
git clone --depth 1 https://github.com/leonardodino/p5-boilerplate.git project-name
cd project-name
yarn
```

## Dev Server

```bash
yarn dev
```

## Compile
```bash
yarn build
```
